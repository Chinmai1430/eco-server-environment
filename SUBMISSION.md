###### Executive Summary

This reinforcement learning environment addresses climate change through AI-powered environmental restoration. Our system achieves an 85% reduction in pollution levels while improving biodiversity scores from 50% to 60%. Deployed live at https://chinmai1430-eco-server-environment.hf.space/



###### 🌍 Project Overview

EcoServer Environment is an innovative OpenEnv reinforcement learning system designed to combat climate change through strategic environmental restoration. The system simulates real-world ecological challenges including pollution management, resource allocation, and biodiversity conservation.

##### 🛠️ Technical Implementation



###### Core Environment Features:

>20x20 Grid-Based Simulation - Realistic ecosystem modeling

>5 Cell Types: Empty land, Forest, Pollution, Industrial, Water

>Resource Management - Strategic allocation of limited resources

>Dynamic Weather System - Wind affects fire spread patterns

>Multi-Agent Coordination - Complex decision-making scenarios



###### AI Agent Strategies Developed:

>🌿 Conservation First Agent - Prioritizes direct pollution removal (🏆 WINNER)

>🏢 Development Focused Agent - Balances economic growth with ecology

>⚖️ Balanced Approach Agent - Adaptive strategy for mixed conditions

>🌱 Smart Eco Agent - General-purpose environmental strategy



###### API Endpoints:

>POST /reset - Initialize environment (returns grid, pollution, biodiversity)

>POST /step - Execute agent actions (plant trees, remove pollution, etc.)

>GET /state - Retrieve current environment state

>GET /health - System health check

>GET / - API documentation and endpoint information
